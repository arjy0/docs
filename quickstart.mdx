Sonarly is an AI-first debugging platform that detects bugs, clusters them intelligently, and generates LLM-ready context to fix them with your coding agent.

## Get started

### Step 1: Sign up

1. Sign up at [sonarly.dev](https://sonarly.dev/onboarding)
2. Link your GitHub or Cursor account

### Step 2: Install Prompt

Setup Sonarly in 3min with our Open Install Prompt in Cursor

- ✅ Install the Sonarly tracker
- ✅ Configure sourcemaps upload
- ✅ Set up backend tracing (if applicable)

### Step 3: Deploy & start catching bugs

Deploy your app. Sonarly will:

- Record user sessions
- Detect errors automatically
- Send you AI-powered alerts
- Generate fix context for your coding agent
