## Frontend tracker

### Option A: NPM package (recommended)

```bash
npm install @sonarly/tracker
```

```typescript
import Tracker from '@sonarly/tracker';

const tracker = new Tracker({
  projectKey: "YOUR_PROJECT_KEY",
  ingestPoint: "https://sonarly.dev/ingest"
});

tracker.start();
```

### Option B: HTML script tag

```html
<script>
  (function(A, opts){
    window.Sonarly = { opts: opts, queue: [] };
    var s = document.createElement('script');
    s.src = A;
    s.async = true;
    document.head.appendChild(s);
  })("https://sonarly.dev/static/tracker.js", {
    projectKey: "YOUR_PROJECT_KEY",
    ingestPoint: "https://sonarly.dev/ingest"
  });
</script>
```

## Troubleshooting

### Tracker not connecting

1. Check `projectKey` is correct
2. Check `ingestPoint` is `https://sonarly.dev/ingest`
3. Enable debug mode: `__debug__: 5`
